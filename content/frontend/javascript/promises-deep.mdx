## Promise States

Three states, settle once, one-way.

```javascript
new Promise((resolve, reject) => {
  /* pending */
  resolve("done") // → fulfilled
  // reject("err") // → rejected
})
new Promise((r) => {
  r("first")
  r("second")
}) // "first" — second ignored
```

## Chaining

Each .then returns new promise. Rejections skip to nearest .catch.

```javascript
fetch("/api/user/1")
  .then((r) => {
    if (!r.ok) throw Error("HTTP " + r.status)
    return r.json()
  })
  .then((user) => fetch("/api/posts?userId=" + user.id))
  .then((r) => r.json())
  .catch((err) => console.error(err))
  .finally(() => hideLoader())
```

<Alert>
  <AlertTriangle />
  <AlertTitle>Gotcha</AlertTitle>
  <AlertDescription>
    Put .catch before .finally. Errors in .finally aren't caught by .catch
    after.
  </AlertDescription>
</Alert>

## Promise.withResolvers()

External resolve/reject — clean for callback wrapping.

```javascript
const { promise, resolve, reject } = Promise.withResolvers()
fs.readFile("data.txt", "utf-8", (err, data) =>
  err ? reject(err) : resolve(data)
)
await promise
```

## See It In Action

<LessonCodeExample
  title="Implement Promise.all"
  code={`function myPromiseAll(iterable) {
  const promises = Array.from(iterable)
  const results = new Array(promises.length)
  let settled = 0

return new Promise((resolve, reject) => {
promises.forEach((p, i) => {
Promise.resolve(p)
.then((val) => {
results[i] = val
settled++
if (settled === promises.length) resolve(results)
})
.catch(reject) // Reject immediately on first failure
})
})
}

// Usage — resolves with [1, 2, 3]
const p1 = Promise.resolve(1)
const p2 = Promise.resolve(2)
const p3 = Promise.resolve(3)
myPromiseAll([p1, p2, p3]).then(console.log)

// Usage — rejects immediately
const failing = Promise.reject("oops")
myPromiseAll([p1, failing, p3]).catch((err) => console.error("Caught:", err))`}
  language="javascript"
  explanation="`myPromiseAll`mirrors`Promise.all`: wraps each value with `Promise.resolve`, collects results in input order, resolves the outer promise when all complete, and rejects immediately on the first rejection — a classic fail-fast pattern."
/>
