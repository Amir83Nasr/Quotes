## Scope Types


## Hoisting & TDZ

```javascript
console.log(a)          // undefined (hoisted)
// console.log(b)       // ReferenceError: TDZ
var a = 1
let b = 2  // hoisted but uninitialized until this line
```

## Closures

A closure remembers its lexical scope even when called elsewhere.

```javascript
function createCounter() {
  let count = 0
  return { increment: () => ++count, getCount: () => count }
}
const c = createCounter()
c.increment()      // 1
// No direct access to count

// Private state
function createBankAccount(initial) {
  let balance = initial
  return {
    deposit: (amt) => { balance += amt; return balance },
    getBalance: () => balance,
  }
}

// Memoization
function memoize(fn) {
  const cache = new Map()
  return (arg) => cache.has(arg) ? cache.get(arg) : (cache.set(arg, fn(arg)), cache.get(arg))
}
```

## Loop + Closure Gotcha

```javascript
for (var i = 0; i < 3; i++)
  setTimeout(() => console.log(i), 0)  // 3, 3, 3 (shared binding)
for (let i = 0; i < 3; i++)
  setTimeout(() => console.log(i), 0)  // 0, 1, 2 (per-iteration binding)
```
